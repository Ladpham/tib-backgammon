# TIB Backgammon

## Client
- React + boardgame.io frontend
- Deployed to GitHub Pages

## Server
- Node.js + boardgame.io server
- Deploy to Railway

## Setup

```bash
git clone https://github.com/Ladpham/tib-backgammon.git
cd tib-backgammon

# Install client deps
npm install

# Install server deps
cd server
npm install
cd ..
